## 📖 Sobre

Horizon é um RPG de fantasia sombria com sistema de batalha por turnos. Você controla o Viajante, um espadachim misterioso de capa vermelha, em uma jornada através de terras esquecidas.

## ✨ Features (v0.7)

- 🗡️ Sistema de batalha por turnos (Ataque, Magia, Cura, Defesa)
- 🎨 Herói em pixel art 48x72 extraído de artwork original
- 🌄 Cenário 2.5D com parallax e partículas
- 📈 Sistema de XP e level up
- 👹 5 tiers de inimigos com scaling
- 💫 Animações de ataque, capa, idle, e efeitos visuais
- 📱 Interface responsiva

## 🛠️ Stack

- HTML5 Canvas
- Vanilla JavaScript
- CSS3 com animações
- Zero dependências externas
